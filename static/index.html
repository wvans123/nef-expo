AF 能力注册 · 双向开放

NEF 不仅把网络能力对外开放，外部 AF 也能把自己的能力（主要以 tool 形式）注册进网络。注册后能力上架能力超市、同时进内网发现目录（GET /internal/v1/third-party-capabilities），被网络内部 Agent 发现并反向调用 —— 反向调用统一经 NEF 出向网关做鉴权、计费、审计：内网 Agent 调用走信任域内身份（免 AF Key），跨 AF 调用走正常北向订阅鉴权。

① 注册我的能力

类型（可选，默认 tool）
类型只影响内网把它归到哪个承接 Agent，不影响 AF 使用，一般保持 tool 即可。
元/次
按次：调用方随调随付、按次计费；包月：调用方需先订阅、按月计费。
