AF 能力注册 · 双向开放

NEF 不仅将网络能力对外开放，外部 AF 也可以把自己的能力（AI 模型 / 工具 / 数据源）注册进网络。注册后能力上架能力超市，被网络内部 Agent 发现并反向调用 —— 调用经过 NEF 出向网关完成鉴权、计费与审计。

AF 能力 ─注册─▶ NEF 能力超市 ─发现─▶ Network Agent ─编排─▶ NEF 出向网关鉴权·计费·审计 ─反向调用─▶ AF Endpoint
